# 🎭 AI-Based Real-Time Emotion Detection System

## 📌 Overview
The AI-Based Real-Time Emotion Detection System is a computer vision application that detects human emotions from live webcam video using artificial intelligence and deep learning. The project uses the DeepFace library with OpenCV to analyze facial expressions in real time and classify emotions such as happiness, sadness, anger, fear, surprise, disgust, and neutrality.

The system provides a modern user interface with emotion probability bars, FPS monitoring, face detection overlays, screenshots, and real-time visualization.

---

## 🚀 Features
- 🎥 Real-time webcam emotion detection
- 😀 Detects multiple emotions instantly
- 📊 Emotion confidence percentage bars
- 🖼️ Screenshot capture support
- ⚡ FPS (Frames Per Second) monitoring
- 🎨 Color-based emotion visualization
- 🔲 Rounded face detection boxes
- 👥 Multi-face support
- 🧠 AI-powered facial analysis using DeepFace
